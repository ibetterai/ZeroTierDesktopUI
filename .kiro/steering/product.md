# Product Overview

## Product Description

ZeroTier Desktop Tray Application is a cross-platform system tray interface for controlling and managing a local ZeroTier service. It provides users with an intuitive graphical interface to manage ZeroTier virtual networks directly from their system tray.

## Core Features

- **System Tray Integration**: Persistent system tray icon with context menu for quick access to network management
- **Network Management**: Join, disconnect, and monitor ZeroTier virtual networks
- **Network Configuration**: Control network settings including:
  - Managed address assignment
  - Global IP assignment
  - Default route override
  - DNS configuration
- **Authentication Support**: SSO (Single Sign-On) authentication for networks requiring it
- **Address & Route Display**: View assigned IP addresses and managed routes for each network
- **Network Persistence**: Remember disconnected networks for quick reconnection
- **Clipboard Integration**: Copy network IDs, addresses, and routes to clipboard
- **Desktop Notifications**: System notifications for network events and authentication requirements
- **Auto-Start Configuration**: Option to launch UI at system login

## Target Use Case

The application targets users who need to manage ZeroTier virtual networks on desktop operating systems (Windows, macOS, and Linux). Specific scenarios include:

- **Remote Workers**: Connecting to corporate networks securely from home
- **System Administrators**: Managing multiple ZeroTier networks across different services
- **Developers**: Accessing development environments across distributed teams
- **Home Users**: Creating secure peer-to-peer networks for personal use

## Key Value Proposition

- **Simplicity**: Single-click network management without command-line complexity
- **Cross-Platform**: Consistent experience across Windows, macOS, and Linux
- **Lightweight**: Minimal resource usage with background priority threads
- **Native Integration**: Uses platform-native UI components and system tray APIs
- **Secure**: Direct communication with local ZeroTier service via authenticated API
- **Self-Contained**: Includes all necessary dependencies for standalone operation
- **Open Source**: Mozilla Public License (MPL) 2.0 allowing community contributions
